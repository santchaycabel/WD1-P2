HTML & CSS Cheat Sheet

HTML Tag/Attribute Description & Sample
<h1>–<h6> Headings = <h1>Title</h1>
<p> Paragraph = <p>Text</p>
<head>, <body> Head: metadata, Body: visible content
<meta> Metadata = <meta charset="UTF-8">
<a> Link = <a href="url">Click</a>
<table> Table = <table><tr><td>Data</td></tr></table>
<ul>, <ol>, <li> Lists = <ul><li>Item</li></ul>
<form> Form = <form><input type="text"></form>
<title> Page title inside <head>
<link rel="icon"> Favicon = <link rel="icon" href="favicon.ico">
<b>, <i>, <u> Text formatting tags
<img> Image = <img src="pic.jpg" alt="desc">
<!-- --> Comment in HTML
<nav>, <aside>, <footer> Layout tags
CSS Topic Description & Example
Selectors p { color: red; }
Comments /* This is a comment */
Colors color: #ff0000; or color: red;
Measurements px, %, em, rem, vh, vw
CSS Property Description & Example
background background-color: yellow;
border border: 1px solid black;
padding / margin padding: 10px; margin: 20px;
opacity opacity: 0.5;
height / width height: 100px; width: 200px;
text & fonts font-size: 16px; font-family: Arial;
icons Font Awesome <i class="fa fa-home"></i>
list & tables list-style-type: square;
position / z-index / overflow / float / inline-block position: absolute; z-index: 2; overflow: hidden; float: left; display: inline-block;
navigation bar & buttons nav a { display: block; background: blue; }
dropdown Use: hover for dropdown menus
image gallery Use CSS Grid or Flexbox
